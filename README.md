# CLI Banking Management System (Python)

A **Command Line Interface (CLI) Banking Management System** built using **Python and JSON for persistent data storage**.
This project simulates basic banking operations such as creating accounts, depositing and withdrawing money, updating account details, and deleting accounts.

The goal of this project is to practice **Object-Oriented Programming (OOP), file handling, and CRUD operations** in Python.

---

## Features

* Create a new bank account
* Deposit money into an account
* Withdraw money from an account
* View account details
* Update account information (name, age, email, pin, phone number)
* Delete an existing account
* JSON-based database for storing account data
* Basic input validation and authentication using account number and PIN

---

## Technologies Used

* **Python**
* **JSON (for data storage)**
* **Object-Oriented Programming (OOP)**
* **File Handling**

---

## How It Works

The program stores all account information inside a JSON file.
Each account contains:

* Name
* Age
* Email
* PIN
* Phone Number
* Account Number
* Balance

Users interact with the system through a **menu-driven CLI interface**, where they can choose different banking operations.
